# OIBSIP - Web Development and Design Internship

## Task 3 - To-Do Web Application

### Objective

Develop a responsive To-Do Web Application that helps users efficiently manage their daily tasks using HTML, CSS, and JavaScript.

### Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* User-friendly and responsive interface

### Technologies Used

* HTML5
* CSS3
* JavaScript

### Outcome

Successfully developed a responsive To-Do Web Application that enables users to organize and manage their daily activities with a simple and interactive interface.
